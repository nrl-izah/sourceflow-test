## Overview
This is a frontend implementation of a job listing page with a hero search bar, latest jobs carousel, and responsive design. The focus is on UI/UX, accessibility, and responsive behavior rather than backend functionality.

## Tech Stack
- **React / Next.js (app router)**
- **TypeScript**
- **Tailwind CSS**
- **Swiper.js** (carousel)
- **Figma designs used as reference** (if applicable)

## Notes
- This project is frontend only. The search input is controlled but not connected to a backend.
- The carousel navigation buttons are fully functional with responsive behavior.
- All colors, spacing, and typography are implemented according to the design spec.
